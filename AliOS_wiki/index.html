Kernel Entry (_start)

The entry point bridges the gap between the bootloader and C code.

section .text
bits 32
global _start
_start:
    cli
    mov esp, stack_top
    call setup_paging
    jmp gdt64.code:long_mode_start

GDT & Segments

The Global Descriptor Table defines memory segments in Long Mode.

gdt64:
    dq 0 ; null descriptor
    dq (1<<43) | (1<<44) | (1<<47) | (1<<53) ; code segment
